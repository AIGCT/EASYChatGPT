## 注意事项

1. 本项目仅用于方便更多开发者体验ChatGPT,生成的内容与本平台无关
2. 暂无实现多轮对话。
3. 有问题可以在评论区讨论
4. 好玩的对话也可以放评论区大家看看，但是禁止搞zz、ys和任何敏感问题。
5. 需要自行更换apikey在config.json中。可以添加多个Key。

## 1 安装环境

# 运行一次即可 安装代码环境

```
pip3 install -r requirements.txt
```

## 2 开始对话！

`python3 app.py`

## ☆PS☆

1. 你要在没有开启代理的情况下运行app.py
